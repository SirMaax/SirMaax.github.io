Soulkeeper

Ein kleines Game welches für einen Gamejam innerhalb von 72 Stunden erstellt wurde, dabei wurde alles bis auf die Hintergrundmusik selbst gemacht. In dem Spiel geht es darum die Seelen im Himmel erfolgreich zu Hüten und dabei von Gefahren und Hindernissen zubewahren.

Amateur Downloader

Ein kleines nerviges und nach Design frustierendes Spiel, welches für einen Gamejam innerhalb von 42 Stunden erstellt wurde, dies geschah in Kooperation mit einem Freund. Musik, Code und Assets wurden alle innerhalb der 42 Stunden erstellt.

In Amateur Downloader ist das Ziel den richtigen Downloadbutton zwischen allen anderen schnell genug zu finden, bevor die Polizei vor dem Haus steht.

Double Density Data Defender

DDDD ist ein Game welches wieder in Kooperation mit Tabzeen für einen Gamejam gemacht wurde.

Es sollte ein Wave-Shooter werden, mit Mechaniken welche den Spieler in verschiedenen Bereichen einschränken, sofern dieser Schaden nimmt. Leider wurde dieses Ziel nicht erreicht. Nicht destotrotz wurden wertvolle Dinge von dem kleinen Projekt gelernt.

  • Itch.io Spiellink

Tower Dice Defense

TDD ist das Spiel welches während meines ersten Gamejam entstanden ist. Viel wurde beim erstellen gelernt

Anstelle des klassischen Tower Defense Spiels unterliegt die Auswahl und Platzierung der Türme anderen Regeln. 3 Würfel werden auf der Karte gerollt, die Werte dieser können dann beliebig addiert und subtrahiert werden um eine Zahl zwischen 1 und 18 zubekommen. Dieser Turm kann dann auf Linien welche die 3 Würfel verbinden platziert werden

  • Itch.io Spiellink
